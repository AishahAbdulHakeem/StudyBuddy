# StudyBuddy

StudyBuddy is a student-matching app concept designed to help Cornell students find study partners based on shared courses, academic interests, and profile information.

## Overview

College students often struggle to find classmates to study with, especially in larger courses or when they do not already know people in the class. StudyBuddy helps make study groups easier to discover by organizing student profiles around courses and shared academic interests.

## Features

- Browse student profiles
- Match with students taking the same classes
- Search or filter by course
- Support academic connection-building within Cornell
- Create a more approachable way for students to form study groups

## Project Goals

The goal of this project is to make academic collaboration more accessible. Instead of relying only on group chats, office hours, or already knowing someone in a class, students can use StudyBuddy to discover classmates who are also looking for study partners.

## What I Built

- Designed the core user flow for finding study partners
- Structured the app around course-based matching
- Created a simple, student-focused concept for academic networking
- Built the foundation for profile browsing and study group discovery
